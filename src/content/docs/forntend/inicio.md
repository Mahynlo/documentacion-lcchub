# Heramintas usadas para el frontend 

## NEXT.JS
si te preguntas que es es si Next.js es un framework de React de código abierto desarrollado por Vercel que facilita la creación de aplicaciones web rápidas y escalables. Ofrece renderizado híbrido (servidor y cliente), optimización automática, enrutamiento basado en archivos y soporte para funciones serverless, lo que mejora el SEO y el rendimiento. Esta por lo menos es una descripcion que se pude encontrar en internet y se recomienda entender como es que fuenciona este framework.

## React 
Si te preguntas que es React es una biblioteca de JavaScript de código abierto, creada por Facebook en 2013, diseñada específicamente para construir interfaces de usuario (UI) interactivas, rápidas y modulares. Se basa en componentes reutilizables y utiliza un Virtual DOM para optimizar el rendimiento al actualizar solo las partes necesarias de la página, ideal para aplicaciones de una sola página (SPA).Esta por lo menos es una descripcion que se pude encontrar en internet y se recomienda entender como es que fuenciona esta biblioteca de JavaScript.
