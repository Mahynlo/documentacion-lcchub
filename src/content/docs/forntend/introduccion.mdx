Bienvenid@ a la guia de mantenimiento de LCC-HUB fue creada con el proepeosito de mantenimeiento 
de la pagina esta seccion es enfoecada al frontend aqui se describe lo necesario para poder enetender

## Conceptos de Herramientas 

Para poder modifcar el frontend de LCC-HUB se requeire algo de conosimientos en Html,javascript, y css tambien 
en el suso de frameworks como lo son next.js y tambien en el suso de biblioteca de javascript como React y deribados 
para ser suadas en el ecosistema de Reat ademas de tener conceptos de lo que es node.js y instaladores de 
paquetes como npm. A continuacion se describen que son exactamentes lo que se necesita saber.

## Heramintas usadas para el frontend 

### -NEXT.JS
Si te preguntas que es Next.js es un framework de React de código abierto desarrollado por Vercel que 
facilita la creación de aplicaciones web rápidas y escalables. Ofrece renderizado híbrido (servidor y cliente), 
optimización automática, enrutamiento basado en archivos y soporte para funciones serverless, lo que mejora
 el SEO y el rendimiento. Esta por lo menos es una descripcion que se pude encontrar en internet y se recomienda 
 entender como es que fuenciona este framework.

### -REACT 
Si te preguntas que es React es una biblioteca de JavaScript de código abierto, creada por Facebook en 2013, diseñada 
específicamente para construir interfaces de usuario (UI) interactivas, rápidas y modulares. Se basa en componentes 
reutilizables y utiliza un Virtual DOM para optimizar el rendimiento al actualizar solo las partes necesarias 
de la página, ideal para aplicaciones de una sola página (SPA).Esta por lo menos es una descripcion
que se pude encontrar en internet y se recomienda entender como es que fuenciona esta biblioteca de JavaScript.

### -NODE.JS
Si te preguntas que es Node.js es un entorno de ejecución de código abierto y multiplataforma, basado 
en el motor V8 de Google Chrome, que permite ejecutar código JavaScript en el servidor, fuera del navegador web. 
Es ideal para aplicaciones escalables, en tiempo real (chats, streaming) y eficientes, utilizando un modelo de 
entrada/salida asíncrono y no bloqueante.Esta por lo menos es una descripcion
que se pude encontrar en internet y se recomienda entender como es que fuenciona el entorno de ejecucion para JavaScript.

### -Tailwind CSS 

Si te preguntas que es Tailwind CSS es un framework de CSS "utility-first" (prioridad a la utilidad) que permite diseñar sitios web personalizados rápidamente directamente en el HTML, utilizando clases predefinidas como flex, pt-4 o text-center. A diferencia de Bootstrap, no proporciona componentes predefinidos, lo que ofrece mayor flexibilidad y un diseño único sin salir de tu archivo de marcado.
:::note
Estos concepto son basicos para entender como esta cosntruido LCC-HUB asi que se suguiere saber que es o tener nociones basicas.
:::
