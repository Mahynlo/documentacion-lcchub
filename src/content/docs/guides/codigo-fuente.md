El ecosistema de la aplicación se divide en dos componentes principales: **Frontend** (interfaz de usuario y documentación) y **Backend** (procesamiento de datos). Para detalles específicos sobre la replicación de entornos locales, consulte el archivo `README.md` de cada repositorio.

## Estructura de Carpetas (Proyecto General)

```text
📦 proyecto-root
 ┣ 📂 src/                # Lógica principal y componentes
 ┣ 📂 public/             # Recursos estáticos (imágenes, iconos)
 ┣ 📜 astro.config.mjs    # Configuración del framework Astro
 ┣ 📜 starlight.config.mjs # Configuración del motor de documentación
 ┗ 📜 package.json        # Gestión de scripts y dependencias
