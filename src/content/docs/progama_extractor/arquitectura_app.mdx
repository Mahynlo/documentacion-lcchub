## Diagrama de flujo de uso 
![Flujo de uso](/images/arquitectura_extractor.png)


En la imagen que se presenta se puede apresiar como es el flujo de uso de la palicacion este flujo de uso de la aplicacion el priencipal en el moduelo de extraccion de datos de la aplicacion que este el el priencipal y mas importante ya que sera usado para extraer datos oficiales de un archivo gigante con todaos los alumnos en la carrera pero nosotros solo necesistamos extraer la informacion de los que estan firmados hasta la fecha.

El **Gestor de Archivos LCC HUB**, una aplicación diseñada para facilitar la gestión, procesamiento y validación de archivos CSV/Excel del sistema LCC Hub.

Esta herramienta te permite:

- ✅ **Gestionar Autorizaciones**: Buscar alumnos y marcar quiénes autorizaron el uso de sus datos personales
- 📂 **Cargar Datos Oficiales**: Importar archivos CSV/Excel con información de alumnos
- 🔧 **Procesar Archivos**: Ordenar, limpiar, comparar y extraer datos de archivos
- 📊 **Validar Estructuras**: Verificar que los archivos cumplan con el formato requerido
