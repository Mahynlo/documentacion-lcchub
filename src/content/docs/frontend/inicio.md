# Inicio
Bienvenid@ a la guía de mantenimiento de LCC-HUB, la cual fue creada con el propósito de dar mantenimiento a la página. Esta sección está enfocada al frontend. Aquí se describe lo necesario para poder entender su estructura, arquitectura y las herramientas usadas en el proceso de construcción y futuras modificaciones de la aplicación.

# Conceptos de Herramientas 

Para poder modificar el frontend de LCC-HUB se requieren conocimientos básicos en HTML, JavaScript y CSS, así como en el uso de frameworks como Next.js y bibliotecas de JavaScript como React y derivados para ser usadas en el ecosistema de React, además de tener conceptos de lo que es Node.js e instaladores de paquetes como npm. A continuación se describe exactamente lo que se necesita saber.

# Herramientas usadas para el frontend 

## NEXT.JS
Si te preguntas qué es, en sí Next.js es un framework de React de código abierto desarrollado por Vercel que facilita la creación de aplicaciones web rápidas y escalables. Ofrece renderizado híbrido (servidor y cliente), optimización automática, enrutamiento basado en archivos y soporte para funciones serverless, lo que mejora el SEO y el rendimiento. Esta es por lo menos una descripción que se puede encontrar en internet y se recomienda entender cómo funciona este framework.

## React 
Si te preguntas qué es React, es una biblioteca de JavaScript de código abierto, creada por Facebook en 2013, diseñada específicamente para construir interfaces de usuario (UI) interactivas, rápidas y modulares. Se basa en componentes reutilizables y utiliza un Virtual DOM para optimizar el rendimiento al actualizar solo las partes necesarias de la página, ideal para aplicaciones de una sola página (SPA). Esta es por lo menos una descripción que se puede encontrar en internet y se recomienda entender cómo funciona esta biblioteca de JavaScript.
